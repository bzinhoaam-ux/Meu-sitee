📖 Conceitos Fundamentais

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar a estrutura de uma página web. É como o "esqueleto" de um site.

O que é CSS?

CSS (Cascading Style Sheets) é a linguagem de estilo usada para estilizar a página. É como a "roupa" e a "decoração" do site.

HTML CSS
Define a estrutura (títulos, parágrafos, imagens) Define a aparência (cores, fontes, tamanhos)
É como os ossos do corpo É como a pele e as roupas
Usa tags como <h1>, <p>, <div> Usa propriedades como color, font-size, margin
🎯 Passo a Passo

Por onde começar?

1. Instale um editor de código

Recomendação: Visual Studio Code (VS Code) — gratuito, leve e muito usado. Baixe em code.visualstudio.com

2. Crie seu primeiro arquivo

Crie uma pasta no seu computador. Dentro dela, crie um arquivo chamado index.html.

3. Estrutura básica de um HTML

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Meu primeiro site</title>
</head>
<body>
    <h1>Olá, mundo!</h1>
    <p>Este é meu primeiro site.</p>
</body>
</html>

✅ Agora abra esse arquivo no navegador. Você já tem uma página web!

🎨 Estilizando

CSS Básico: Como estilizar

3 formas de usar CSS

  • Inline: dentro da própria tag (menos recomendado)
  • Interno: dentro da tag <style> no cabeçalho (bom para testes)
  • Externo: em um arquivo .css separado (recomendado para projetos)

Exemplo de CSS Interno

<style>
    body {
        font-family: Arial, sans-serif;
        background-color: #f0f0f0;
    }
    h1 {
        color: #2a5298;
        text-align: center;
    }
    p {
        font-size: 18px;
        line-height: 1.5;
    }
</style>

Exemplo de CSS Externo (arquivo style.css)

/* style.css */
body {
    margin: 0;
    padding: 0;
    background: #f4f4f4;
}
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}
.btn {
    background: #e67e22;
    color: white;
    padding: 10px 20px;
    border-radius: 5px;
    text-decoration: none;
}
.btn:hover {
    background: #d35400;
}

🏗️ Projeto Prático

Passo a passo: Criando um site completo

Passo 1: Estrutura HTML

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Site Completo</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Meu Site</h1>
        <nav>
            <a href="#">Início</a>
            <a href="#">Sobre</a>
            <a href="#">Contato</a>
        </nav>
    </header>
    <main>
        <section class="hero">
            <h2>Bem-vindo!</h2>
            <p>Este é um site completo criado com HTML e CSS</p>
            <a href="#" class="btn">Saiba mais</a>
        </section>
        <section class="cards">
            <div class="card">
                <h3>HTML</h3>
                <p>Linguagem de marcação para estrutura</p>
            </div>
            <div class="card">
                <h3>CSS</h3>
                <p>Linguagem de estilo para design</p>
            </div>
        </section>
    </main>
    <footer>
        <p>&copy; 2025 - Meu Site</p>
    </footer>
</body>
</html>
